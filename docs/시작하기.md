## 5분 빠른 환경 설정

```bash
# 1. 의존성 설치
npm install

# 2. Prisma Client 생성
npx prisma generate

# 3. 환경변수 확인
# .env.local 파일에 DATABASE_URL 설정 확인

# 4. 개발 서버 실행
npm run dev
```

**접속**: http://localhost:3000
