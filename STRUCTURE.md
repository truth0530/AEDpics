# AEDpics 프로젝트 구조

AEDpics 프로젝트의 메인 Next.js 애플리케이션은 `aed-check-system` 디렉토리에 위치합니다.

## 구조

```
AEDpics/
├── aed-check-system/          # Next.js 앱 (독립적 Git 저장소)
├── MIGRATION_PLAN_CAFE24.md
├── CAFE24_SIGNUP_GUIDE.md
├── README.md
└── .gitignore
```

## 참고

`aed-check-system` 폴더는 독립적인 Git 저장소입니다.
`cd aed-check-system`으로 이동하여 해당 저장소의 커밋 이력을 확인할 수 있습니다.
